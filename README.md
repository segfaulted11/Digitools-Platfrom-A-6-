## Project Name : Digital Tool's Platform
## Description : this is a react project.
## Tech Stack 
- HTML
- CSS
- JAVASCRIPT
- REACT JS
- TAILWIND CSS
## Dependencies 
- DAISY UI
- REACT-LUCIDE
- REACT-TOASTIFY
## Features
1) renders JSON Data on the UI
2) Adds products to carts, shows the number of products added to cart and a product can be removed once its added.
3) uses modern and efficient technologies like tailwind, daisyUI, react toaster etc.
